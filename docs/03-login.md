# การเข้าสู่ระบบ

## ขั้นตอนการ Login

1. เปิดเบราว์เซอร์ไปที่ **http://localhost:3000**
2. ระบบจะพาไปหน้า Login โดยอัตโนมัติ
3. กรอก **Username** และ **Password**
4. กด **เข้าสู่ระบบ**

> **บัญชีทดสอบ:** username `demo` / password `demo123`

## Token และ Session

- ระบบใช้ **JWT Token** เก็บใน LocalStorage ของเบราว์เซอร์
- Token หมดอายุหลัง **7 วัน** (ค่า default)
- เมื่อ Token หมดอายุ ระบบจะ redirect กลับหน้า Login อัตโนมัติ

## การออกจากระบบ

คลิกที่เมนูผู้ใช้มุมขวาบน แล้วเลือก **ออกจากระบบ**

## ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| Login ไม่ได้ | ตรวจสอบว่า Backend API รันอยู่ที่ port 5001 |
| หน้าเว็บโหลดไม่ขึ้น | ตรวจสอบว่า Frontend รันอยู่ที่ port 3000 |
| ลืมรหัสผ่าน | ติดต่อ Admin เพื่อ reset รหัสผ่านในฐานข้อมูล |

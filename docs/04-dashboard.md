# Dashboard

หน้าหลักหลังจาก Login สำเร็จ แสดงภาพรวมการใช้งานทั้งหมด

## ส่วนประกอบของหน้า Dashboard

### 1. สถิติการใช้งาน (Stats)

| ช่อง | คำอธิบาย |
|---|---|
| แบบร่างที่บันทึก | จำนวน Draft ที่บันทึกไว้ (สูงสุด 5 Slot) |
| กระเบื้องในคลัง | จำนวนรายการกระเบื้องที่เพิ่มเข้าคลัง |
| บันทึกล่าสุด | วันที่บันทึกงานล่าสุด |

### 2. เริ่มต้นใช้งาน (Quick Actions)

ปุ่มลัดไปยังฟีเจอร์หลัก 3 อย่าง:

- **Floor Planner** → เปิดหน้าออกแบบห้อง 3D
- **คำนวณกระเบื้อง** → เปิดเครื่องคิดเลข
- **คลัง Tile** → เปิดหน้าจัดการกระเบื้อง

### 3. แบบร่างที่บันทึกไว้ (Saved Designs)

แสดงรายการ Slot ทั้ง 5 พร้อมสถานะ:
- Slot ที่มีข้อมูล → แสดงชื่องานและวันที่บันทึก พร้อมปุ่ม **เปิด Planner**
- Slot ว่าง → แสดง "ยังไม่มีข้อมูล"

## การเปิดงานที่บันทึกไว้

คลิกปุ่ม **เปิด Planner →** ที่ Slot ต้องการ ระบบจะเปิด Planner พร้อมโหลดงานออกแบบนั้นทันที

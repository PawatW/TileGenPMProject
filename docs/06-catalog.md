URL: `/planner/catalog`

## เพิ่มกระเบื้องส่วนตัว (Custom Tile)

1. คลิกแท็บ **กระเบื้องพื้น**
2. คลิก **+ เพิ่มกระเบื้อง**
3. กรอกข้อมูล:
   - **ชื่อ** — ชื่อกระเบื้อง
   - **ความกว้าง / ความยาว** — ขนาด (cm หรือ นิ้ว)
   - **แผ่น / กล่อง** — จำนวนแผ่นต่อกล่อง
   - **ราคา / กล่อง** — ราคาในหน่วยบาท
   - **รูปภาพ** — อัปโหลดไฟล์รูป (รองรับ JPG, PNG, HEIC ขนาดไม่เกิน 15 MB)
4. กด **บันทึก**

> กระเบื้อง Custom จะถูกบันทึกลงฐานข้อมูลและ sync ข้ามอุปกรณ์

## เพิ่มผนังส่วนตัว (Custom Wall)

1. คลิกแท็บ **ผนัง**
2. คลิก **+ เพิ่มผนัง**
3. กรอกชื่อและอัปโหลดรูป Texture
4. กด **บันทึก**

## เพิ่มอุปกรณ์ส่วนตัว (Custom Fixture)

1. คลิกแท็บ **อุปกรณ์**
2. คลิก **+ เพิ่มอุปกรณ์**
3. กรอกชื่อ, ขนาด (กว้าง × สูง × ลึก), และเลือกสี
4. กด **บันทึก**

## ลบรายการ

คลิกไอคอน **ลบ (ถังขยะ)** ที่รายการที่ต้องการ ระบบจะขอยืนยันก่อนลบ

> **หมายเหตุ:** รายการ Built-in ไม่สามารถลบได้

# ภาพรวมของระบบ

Studio PM Floor Planner คือเว็บแอปพลิเคชันสำหรับออกแบบพื้นและผนังห้องในรูปแบบ 3D พร้อมระบบคำนวณกระเบื้องและสร้างใบเสนอราคาอัตโนมัติ

## ฟีเจอร์หลัก

| ฟีเจอร์ | คำอธิบาย |
|---|---|
| Floor Planner | ออกแบบห้องแบบ 3D เลือกลาย/สีกระเบื้องพื้นและผนัง วางประตู-หน้าต่าง |
| คลัง Tile | จัดการรายการกระเบื้องส่วนตัว เพิ่ม/ลบ/อัปโหลดรูป |
| คำนวณกระเบื้อง | คำนวณจำนวนแผ่น กล่อง และราคารวมสุทธิ พร้อมแผนผังการปู |
| ใบเสนอราคา | สร้างและบันทึก Quotation ส่งออกเป็น PDF ได้ |
| บันทึกแบบร่าง | บันทึกงานออกแบบได้สูงสุด 5 Slot ต่อบัญชี |

## สถาปัตยกรรมของระบบ

```
Browser (Next.js 14)
      │
      ├── /dashboard      → หน้าหลัก
      ├── /planner        → 3D Floor Planner (Three.js)
      ├── /planner/catalog → คลัง Tile
      └── /calculator     → คำนวณกระเบื้อง
      │
      ▼
Flask API (port 5001)
      │
      ▼
PostgreSQL Database
```

## Port ที่ใช้งาน

| Service | URL |
|---|---|
| Frontend | http://localhost:3000 |
| Backend API | http://localhost:5001 |
| PostgreSQL | localhost:5432 |

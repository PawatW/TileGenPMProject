# ใบเสนอราคา (Quotation)

ระบบรองรับการสร้างและบันทึกใบเสนอราคาจาก Floor Planner โดยตรง

---

## การสร้างใบเสนอราคา

1. เปิด **Floor Planner** และออกแบบห้องเสร็จสิ้น
2. คลิกปุ่ม **สร้างใบเสนอราคา** (ใน Toolbar)
3. กรอกข้อมูล:
   - **ชื่อลูกค้า**
   - **ชื่อโปรเจกต์**
4. ระบบคำนวณและแสดงสรุป:
   - ลายกระเบื้องพื้น / ผนัง
   - ขนาดห้อง (กว้าง × ลึก × สูงผนัง)
   - พื้นที่รวม (ตร.ม.)
   - จำนวนแผ่น / กล่อง
   - ราคารวม
5. กด **บันทึก** เพื่อบันทึกลงระบบ

---

## ข้อมูลในใบเสนอราคา

| ช่อง | คำอธิบาย |
|---|---|
| ชื่อลูกค้า | ชื่อผู้รับใบเสนอราคา |
| ชื่อโปรเจกต์ | ชื่องาน |
| ลายกระเบื้องพื้น | Pattern ที่เลือกใน Planner |
| ลายผนัง | Pattern ผนังที่เลือก |
| ขนาดห้อง | กว้าง × ลึก เมตร |
| ความสูงผนัง | เมตร |
| พื้นที่รวม | ตร.ม. |
| จำนวนแผ่น | แผ่น |
| จำนวนกล่อง | กล่อง |
| ราคา/กล่อง | บาท |
| ราคารวม | บาท |

---

## การส่งออกเป็น PDF

1. เปิดรายการ Quotation ที่บันทึกไว้
2. คลิก **ดาวน์โหลด PDF**
3. ไฟล์ PDF จะถูกดาวน์โหลดทันที

> PDF ถูกสร้างและเก็บไว้ในฐานข้อมูลในรูปแบบ Base64

---

## การดูรายการ Quotation

ระบบเก็บ Quotation ล่าสุด **100 รายการ** ต่อบัญชี เรียงตามวันที่สร้างล่าสุดก่อน

---

## API Endpoints (สำหรับนักพัฒนา)

| Method | Endpoint | คำอธิบาย |
|---|---|---|
| `GET` | `/api/quotations` | ดึงรายการ Quotation ทั้งหมด |
| `POST` | `/api/quotations` | สร้าง Quotation ใหม่ |
| `GET` | `/api/quotations/{id}` | ดึงข้อมูล Quotation พร้อม PDF data |

ทุก Endpoint ต้องแนบ JWT Token ใน Header:
```
Authorization: Bearer <token>
```
